### Prerequisites

- **npm** (v6 or higher)

### Installation

1. **Clone the repository**
   ```bash
   git clone git@github.com:tanmay2301/CRM-dummy.git
   ```
2. **Navigate into the project folder**
   ```bash
   cd launchsync
   ```
3. **Install dependencies**
   ```bash
   npm install
   ```
4. **Start the development server**
   ```bash
   npm start
   ```
